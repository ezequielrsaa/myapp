# CakePHP 3 Shopping Cart

## Install

```bash
composer create-project --prefer-dist -s dev andraskende/cakephp-3-shopping-cart myapp
```

## Create Mysql Database

```bash
mysql -u db_username -p db_password db_name < config/schema/shop.sql
```
